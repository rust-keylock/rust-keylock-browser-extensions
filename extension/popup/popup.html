Rust Keylock Password Manager

Available Passwords

The rust-keylock extension offers passwords that are saved in the rust-keylock application to be used in Web pages.

When you visit a Web page where you need to provide credentials, right click on the input/password field. Any password entries that are created for the URL of this Web page will be offered to fill-in.

Additionally, you may also select any password entries from the list below so that they are always offered for filling-in, regardless if the visited page URL matches the URL defined in the password entry.

Ticking a password will make it always available in the rust-keylock context menu
