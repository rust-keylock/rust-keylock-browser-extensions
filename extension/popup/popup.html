Rust Keylock Password Manager

  • Passphrase configured Open the configuration screen of the rust-keylock application, copy the "Browser Excension Passphrase" (generate one if empty) and paste it in the rust-keylock browser extension preferences.
  • Communication with the rust-keylock app initialized Make sure the rust-keylock application is running and is unlocked.
  • Session with the rust-keylock app is valid Make sure the rust-keylock application is running and is unlocked. Then, press the link below to re-establish connection.
  • Communication with rust-keylock app is healthy Make sure the rust-keylock application is running and is unlocked. Then, press the link below to re-establish connection.

In case any of the above is not Ok, please make sure that the rust-keylock application is running and is unlocked and click here to retry establishing connection.

Tip: hovering any errors reveals a tooltip with details.
